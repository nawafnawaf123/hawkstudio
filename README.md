# Hawk Studio

واجهة تعريفية ثابتة لـ **Hawk Studio** مبنية بـ Next.js.

لا يحتوي الموقع على قاعدة بيانات أو لوحة إدارة أو API أو نظام مصادقة. المحتوى ثابت والتواصل مباشر عبر البريد والهاتف وواتساب.

## التشغيل

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```
